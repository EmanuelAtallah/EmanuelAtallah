Features

This project is a fully functional Tic-Tac-Toe game designed for two players. The game includes automatic win and draw detection, turn-based gameplay, and input validation to prevent invalid moves. A reset feature allows players to quickly start a new match without restarting the application as well as a score tracking system to keep track of wins, losses, and draws for both players.

The Code

The project was developped using methods, loops, and conditional statements. A 2D array was used to store the game board, while custom methods handled player turns and win detection. I also implemented logic to check rows, columns, and diagonals for a win after a minimum of 5 moves. One challenge was preventing players from overwriting existing spaces, which I solved through input validation.

How It Turned Out

Overall, the project turned out successfully and helped strenghten my understanding of game logic and program structure. I improved my skills with conditional logic, arrays, and debugging.
